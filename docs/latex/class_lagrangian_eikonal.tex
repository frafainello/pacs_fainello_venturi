class Lagrangian\+Eikonal$<$ P\+H\+D\+I\+M, I\+N\+T\+R\+I\+N\+S\+I\+C\+\_\+\+D\+I\+M $>$}

Implementation of the Lagrangian Eikonal method. 

This class solves the Eikonal equation using the Alternating Direction Method of Lagrange Multipliers (A\+D\+MM). It overrides the necessary virtual methods to provide the specific implementations for computing both the stiffness and reaction terms, and for updating the Lagrangian multipliers.
