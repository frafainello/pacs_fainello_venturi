\hypertarget{index_autotoc_md2}{}\doxysection{Project Overview}\label{index_autotoc_md2}
This project focuses on developing efficient, implicit numerical methods for solving the {\bfseries{Eikonal equation}}, a fundamental equation in modeling wave propagation and computing distance functions. The Eikonal equation has broad applications across fields such as\+: seismology, geospatial physics and computer vision.

The {\bfseries{Eikonal Equation}} is given by\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\(\backslash\)|\(\backslash\)nabla u(x)\(\backslash\)| = F(x)}
\end{DoxyCode}


Where\+:
\begin{DoxyItemize}
\item ( u(x) ) is the distance function.
\item ( F(x) ) represents the wavefront propagation speed.
\end{DoxyItemize}

To solve this, we derive the following {\bfseries{variational formulation}}\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{E(u(x)) = \(\backslash\)int\_\{\(\backslash\)Omega\} \(\backslash\)left( \(\backslash\)|\(\backslash\)nabla u(x)\(\backslash\)| -\/ 1 \(\backslash\)right)\string^2 \(\backslash\), dx}
\end{DoxyCode}


This energy functional is minimized to compute the solution for the Eikonal equation.\hypertarget{index_autotoc_md3}{}\doxysection{Problem Statement}\label{index_autotoc_md3}
This project addresses the numerical solution of this equation by employing {\bfseries{energy minimization principles}} and applying {\bfseries{iterative methods}} such as\+:
\begin{DoxyItemize}
\item {\bfseries{Method of Laplacian Iterations}}
\item {\bfseries{Relaxation of Laplacian Iterations}}
\item {\bfseries{Alternating Direction Method of Lagrange Multipliers (A\+D\+MM)}}
\end{DoxyItemize}

These methods allow for efficient, scalable solutions applicable to a variety of complex scenarios.

\begin{DoxyItemize}
\item {\bfseries{1}}\+: Dirichlet Null B\+Cs on the bottom face (z=0) ~\newline

\item {\bfseries{2}}\+: Dirichlet Null B\+Cs on the central point of the bottom face (z=0, x=0.\+5, y=0.\+5) ~\newline

\item {\bfseries{3}}\+: Dirichlet Null B\+Cs on one vertex (x=0, y=0, z=0) ~\newline

\item {\bfseries{4}}\+: Dirichlet Null B\+Cs on the whole boundary ~\newline

If no input is provided, {\bfseries{Option 1}} is used by default.

e. {\bfseries{Method Selection}}\+: ~\newline

\begin{DoxyItemize}
\item {\bfseries{1}}\+: Standard Eikonal (Method of Laplacian Iterations)
\item {\bfseries{2}}\+: Penalty Eikonal (Relaxation of Laplacian Iterations)
\item {\bfseries{3}}\+: Lagrangian Eikonal (Alternating Direction Method of Lagrange Multipliers)
\end{DoxyItemize}

If no input is provided, {\bfseries{Option 1 (Standard Eikonal)}} is selected by default.
