Detailed Description

template<std::size_t PHDIM, std::size_t INTRINSIC_DIM = PHDIM>
class EikonalEquation< PHDIM, INTRINSIC_DIM >

Abstract base class for solving the Eikonal equation.

This class defines the general framework for solving the Eikonal equation using finite element methods in a parallelized MPI environment. It provides the core functionalities and methods that need to be overridden by derived classes to implement specific solution techniques, such as Standard Eikonal, Penalty Eikonal, and Lagrangian Eikonal methods.

Template Parameters
PHDIMThe physical dimension of the problem.
INTRINSIC_DIMThe intrinsic dimension of the problem (default is equal to PHDIM).
