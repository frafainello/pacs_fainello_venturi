Detailed Description

template<std::size_t PHDIM, std::size_t INTRINSIC_DIM = PHDIM>
class InitialConditions< PHDIM, INTRINSIC_DIM >

This class provides methods for setting initial conditions and solving equations for a finite element mesh.

The class is templated to handle dimensions PHDIM and INTRINSIC_DIM, allowing for flexibility in different physical and geometric configurations. It includes methods for computing initial conditions, such as those for solving the heat equation.

Template Parameters
PHDIMThe physical dimension of the problem.
INTRINSIC_DIMThe intrinsic dimension of the elements in the mesh (defaults to PHDIM).

Solves the heat equation using a sparse linear solver.

This method computes the solution for a heat equation by solving a linear system of equations, taking into account boundary conditions provided as input. The solution is obtained using LU decomposition for sparse matrices.

Parameters
