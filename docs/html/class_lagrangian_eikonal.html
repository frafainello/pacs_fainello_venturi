Detailed Description

template<std::size_t PHDIM, std::size_t INTRINSIC_DIM = PHDIM>
class LagrangianEikonal< PHDIM, INTRINSIC_DIM >

Implementation of the Lagrangian Eikonal method.

This class solves the Eikonal equation using the Alternating Direction Method of Lagrange Multipliers (ADMM). It overrides the necessary virtual methods to provide the specific implementations for computing both the stiffness and reaction terms, and for updating the Lagrangian multipliers.

Template Parameters
PHDIMThe physical dimension of the problem.
INTRINSIC_DIMThe intrinsic dimension of the problem (default is equal to PHDIM).
