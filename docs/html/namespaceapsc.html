Compute the local stiffness matrix if you already have the gradients of the shape functions.

It computes the local stiffness matrix for a linear finite element on an N dimensional simplex given the gradients of the shape functions obtained by computeGradCoeff, the measure of the simplex, and a possible constant elemental multiplicative factor.

Template Parameters
NThe dimension of the space
Parameters
gradCoeffThe gradients of the shape functions computed with computeGradCoeff
measureThe measure of the simplex
factorThe constant multiplicative factor
Returns
The elemental stiffness matrix
Note
This version is useful if you have already computed the gradients of the shape functions and you want to avoid recomputing them Note that the matrix gradCoeff contains the gradients of the shape functions 1 to N in its columns, excluding the gradient of shape function 0 since the latter is just the negative sum of the other gradients
